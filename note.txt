سحب المشروع من القت هاب 

git clone https://github.com/fakhraldinyahya/AraSenttt.git

فتح مجلد المشروع 
فتح ترمنل جديده

إنشاء بيئة افتراضية

نفذ الأمر التالي:

python -m venv venv

 تفعيل البيئة الافتراضية

venv\Scripts\activate


تثبيت المكتبات المطلوبة


pip install -r requirements.txt

نسخ المودلز

تنفذ المشروع 

python app.py

فتح المتصفح قوقل كروم او اي متصفح اخر 
وافتح الرابط التالي :
http://127.0.0.1:5000
راح يعرض واجهه الموقع


ملاحظة:
في كل مرة تغلق فيها موجه الأوامر وتعود للعمل على المشروع،
يجب تفعيل البيئة مرة أخرى باستخدام:

venv\Scripts\activate

ثم تشغيل المشروع هكذا 

python app.py
